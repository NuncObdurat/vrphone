An open source Unity asset package that contains a VR Phone application with some interesting experimental features. 
Use can be found here: https://youtube.com/shorts/OtbSow4zdvQ

The included Unity package contains a base scene that contains the VR Phone application. You will need to import/do the following:

SteamVR (via the Unity Package Manager)
Add your own videos/screen content to the VRPhone.cs script
